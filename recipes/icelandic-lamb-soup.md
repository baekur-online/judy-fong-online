# Kjötsúpa - Icelandic lamb soup

## Ingredients
- [ ] lamb sliced into soup cuts with bone
- [ ] súpujurtir
- [ ] rutabaga, optional
- [ ] two dried chili peppers
- [ ] instant mashed potatoes powder
- [ ] carrots, optional
- [ ] salt
- [ ] pepper
- [ ] pasta spices, optional
- [ ] water
- [ ] potato, optional
